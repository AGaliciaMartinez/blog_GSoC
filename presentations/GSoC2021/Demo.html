Google Summer of Code 2021: qutip-tensorflow

Asier Galicia

qutip-tensorflow

A plug-in for QuTiP providing a TensorFlow linear-algebra backend.

  • We can now operate in a GPU using QuTiP.
  • QuTiP now benefits from TensorFlow’s auto differentiation and optimization features.

Benchmark

Small set of benchmarks can be run locally. Useful to asses if speedups can be achieved.
